= Asset monitoring

Serves **BullionVault** (precious metals) and **Coinbase** (BTC, ETH) spot data as **Prometheus** text at `GET /metrics`. Upstream APIs are queried on a **background schedule**; `/metrics` reads the last successful scrape (default: refresh every 3600s).

* `/metrics` – Prometheus
* `/health` / `/ready` – probes

== Run

```
bundle install
rake server
```

`rake server` and `rake dev` bind **`0.0.0.0:8080`**. `rake dev` uses `rerun` for auto-reload.

[cols="1,2",options="header"]
|===
|Variable |Purpose

|`RACK_ENV` |`production` / `development` (tests force `test` and disable the scraper)
|`PORT` |Listen port (default `8080`)
|`LOG_LEVEL` |Default `INFO`
|`APP_VERSION` |Label on `asset_monitoring_app_info` (default `unknown`)
|`METRICS_SCRAPE_INTERVAL_SECONDS` |Seconds between background scrapes (default `3600`; `0` = only initial scrape, no loop)
|`METRICS_SCHEDULER_DISABLED` |`1` – do not start the background scraper
|===
